# OPUS INTERIORS - Premium Turnkey Interior Solutions

One step for complete interior. OPUS INTERIORS offers end-to-end interior design and execution services in Hyderabad. From design to delivery, we handle everything.

## Getting Started

Follow these steps to set up and run the project locally:

### 1. Prerequisites

Ensure you have Node.js and npm installed.

### 2. Installation

Clone the repository and install the dependencies:

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd opus-interiors-complete-trusted

# Install dependencies
npm install
```

### 3. Development

Start the development server:

```sh
npm run dev
```

The application will be available at `http://localhost:8080` (or the port specified in the terminal).

## Technologies Used

- **Vite**: Next Generation Frontend Tooling
- **TypeScript**: Typed JavaScript
- **React**: A JavaScript library for building user interfaces
- **shadcn-ui**: Reusable components built with Radix UI and Tailwind CSS
- **Tailwind CSS**: A utility-first CSS framework
- **Lucide React**: Beautiful & consistent icons
- **React Router**: Declarative routing for React

## Project Structure

- `src/components`: Reusable UI components
- `src/pages`: Functional page components
- `src/hooks`: Custom React hooks
- `src/lib`: Utility functions and library configurations
- `public`: Static assets

## Deployment

The project can be built for production using:

```sh
npm run build
```

The output will be in the `dist` folder, which can be deployed to any static hosting service.
